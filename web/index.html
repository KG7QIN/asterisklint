Asterisklint Example

Quickly check your Asterisk dialplan.

Paste your dialplan below and click on Evaluate.

Beware! Be sure not to upload sensitive data like passwords; the evaluation is done server-side. And if you want to check larger dialplans, with #include's you should use the asterisklint CLI interface instead; fetch it from github or through pip.
